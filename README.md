<h1 align="center">📚 Library Management System</h1>

<p align="center">
  A complete Library Management System built with <strong>⚛️ React</strong>, 
  <strong>🎨 Tailwind CSS</strong>, and <strong>📂 Redux Toolkit + JSON Server</strong>.
</p>

<hr/>

<h2>✨ Features</h2>
<ul>
  <li>👥 <strong>Member Management</strong> – Add, edit, view, and delete members.</li>
  <li>📚 <strong>Book Management</strong> – Track title, author, genre, ISBN, availability.</li>
  <li>🔄 <strong>Loan System</strong> – Borrow and return books with due dates.</li>
  <li>⏳ <strong>Overdue Tracking</strong> – Auto-detect late returns + calculate fines.</li>
  <li>💰 <strong>Fine System</strong> – Damage fine ₹50, late fine ₹20/day + history.</li>
  <li>📊 <strong>Reports</strong> – Member activity, borrowed history, fine reports.</li>
  <li>🎨 <strong>Modern UI</strong> – Fully responsive Tailwind CSS design.</li>
</ul>

<h2>⚡ Installation & Setup</h2>

<h3>1. Clone the repo 📥</h3>
<pre>
git clone https://github.com/your-username/library-management-system.git
cd library-management-system
</pre>

<h3>2. Install dependencies 📦</h3>
<pre>npm install</pre>

<h3>3. Run JSON Server (Mock API) 🗄️</h3>
<pre>json-server --watch db.json</pre>

<h3>4. Start the React app 🚀</h3>
<pre>npm run dev</pre>

<p>App will run at <code>http://localhost:3000</code></p>
